# Frappe-installer script

Da die Installation von Frappe-Bare-Bones relativ komplex ist und ich mich jedesmal damit abmühe habe ich mit Ki mal ein Installationsskript zusammengestellt, welches den Prozess vereinfachen soll.

eigentlich recht einfach. Skript ziehen, mit "chmod +x frappe-installer.sh" ausführbar machen und dann mit "bash frappe-installer.sh" ausführen. durch die Einzelnen Schritte gehn und sich freuen das es einfach funktioniert.
